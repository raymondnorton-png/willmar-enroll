You are connected to the Enroll network (password: enroll2526)

Step 1: Download & Install Certificate

📥 Download Willmar RADIUS Certificate

After downloading: Open the file and tap Install (choose Wi-Fi or VPN & apps if prompted).

Step 2: Connect to Staff / Student Network

On Android:

  1. Forget the "Enroll" network
  2. Go to Wi-Fi settings and select your main Staff or Student network
  3. Choose WPA2-Enterprise (or WPA3-Enterprise)
  4. EAP method: PEAP
  5. Phase 2 authentication: MSCHAPv2
  6. CA certificate: Select the certificate you just installed (or "Do not validate" / "Default")
  7. Identity / Username: username@willmar.k12.mn.us
  8. Password: Your Willmar password
  9. Tap Connect

On Chromebooks:

  1. Click the clock → Settings → Network → Wi-Fi
  2. Select your main Staff or Student network
  3. Use the same settings as above

After connecting successfully: Forget the Enroll network.

If you run into issues, contact tech support.
