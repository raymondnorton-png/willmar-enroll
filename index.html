Step 1: Install the School Certificate

📥 Download Willmar RADIUS Certificate

How to Install on Android:

  1. Open the Files app → Downloads
  2. Tap Willmar-RADIUS-Server.cer
  3. Choose CA certificate (or Trusted credentials)
  4. Tap Install

If you get a "private key needed" error, go to Settings → Security → Encryption & credentials → Install a certificate → CA certificate, then select the file.

Step 2: Connect to the Secure Network

  1. Forget the "Enroll" network
  2. Select your Staff or Student network
  3. Use PEAP + MSCHAPv2 and select the certificate you installed
  4. Enter your Willmar username and password

You're done!
