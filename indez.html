Step 1: Install the School Root Certificate

📥 Download Willmar Root CA Certificate

How to Install on Android / Pixel:

  1. Open the Files app → Downloads
  2. Tap Willmar-Root-CA.cer
  3. Choose CA certificate (or Trusted credentials)
  4. Tap Install

If you get "private key required", go to:
Settings → Security → Encryption & credentials → Install a certificate → CA certificate, then select the file.

Step 2: Connect to the Secure Network

  1. Forget the "Enroll" network
  2. Select your Staff or Student network
  3. Security: WPA2-Enterprise
  4. EAP method: PEAP
  5. Phase 2 authentication: MSCHAPv2
  6. CA Certificate: Select the one you just installed (or Default)
  7. Username: yourusername@willmar.k12.mn.us
  8. Password: Your Willmar password
  9. Tap Connect

You're done! Forget the Enroll network after connecting.
